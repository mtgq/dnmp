# 用于 PHP 开发的 Docker 环境

在 mac 上构建 docker 环境用于日常编程开发。

* 系统：macOS Sequoia 15.5
* 设备：MacBook M4 Pro
* 工具：OrbStack

## 重建容器
```
docker-compose build --no-cache {container-name}
```
